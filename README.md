### 八种立体声录音话筒制式 [blog](https://www.bilibili.com/opus/599836146181472419)
+ 位置 (1)：AB制式立体声拾音
  + 特点：使用一对全指向性麦克风，距离较远（如3米）以增强立体感。
  + 场景：适合录制大型乐队或合唱团，能够捕捉场地的空间感和声场宽度。
  + 例子：在一场交响乐演奏中，麦克风分别放置在舞台中心稍偏两侧，用于拾取整体的声场和乐队的空间感。
+ 位置 (2)：ORTF制式立体声拾音
  + 特点：两支心形指向麦克风间距17厘米，夹角110°。适合较窄但精准的立体声拾音。
  + 场景：适合中小型乐队或独奏表演，在空间感和声音的清晰度之间找到平衡。
  + 例子：录制一个弦乐四重奏时，麦克风放在乐手前方1.4米处，能够平衡地拾取每位乐手的声音，同时保留一定的立体声效果。
+ 位置 (3)：XY制式立体声拾音
  + 特点：两支心形指向麦克风振膜重叠，夹角90°，主要捕捉声源的方向性。
  + 场景：适合近距离录音，立体声效果较弱，但声音的方向性和定位感较好。
  + 例子：在录制钢琴独奏时，将麦克风放在钢琴正上方，拾取精准的音符细节和方向感。
+ 位置 (4)：多点拾音方式
  + 特点：使用多支心形指向麦克风，分别放置在乐队中各声部的附近，单独拾取每个声部的直达声。
  + 场景：适合分轨录音，需要后期混音调整每个声部的比例。
  + 例子：在一场录制中，将麦克风分别靠近弦乐组、木管组和打击乐组，确保每个组的声音都能独立清晰地录制下来。
